# Absensi Kelas

Vite + React + TypeScript project for class attendance management.

## Requirements
- Bun (recommended)

## Setup
Install dependencies:
```bash
bun install
```

Run the dev server:
```bash
bun run dev
```

Build for production:
```bash
bun run build
```

Preview production build:
```bash
bun run preview
```

## Vercel
This repo is configured to use Bun on Vercel via `vercel.json`:
- Install: `bun install`
- Build: `bun run build`

If you change package managers, update or remove `vercel.json` accordingly.
